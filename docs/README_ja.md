> ## インターフェイス パラメータ

- メソッド: `GET` / `POST`
- ルート: `/push`

| 必要?      | パラメータ | 説明                   |
| ---------- | :--------- | ---------------------- |
| 必ず       | to         | 受信者のメールアドレス |
| オプション | msg        | メール本文             |
| オプション | title      | 件名                   |
| オプション | name       | 差出人のニックネーム   |

> ## 展開

プロジェクトのルート ディレクトリを入力して依存関係をインストールします

```
pnpm i
```

スタート

```
pnpm start
```
